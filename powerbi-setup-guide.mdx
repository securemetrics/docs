### Microsoft PowerBI Desktop App

<AccordionGroup>
  <Accordion title="Download PowerBI Desktop" icon="copy">
    [You can download PowerBI Desktop from Microsoft.](https://www.microsoft.com/en-us/download/details.aspx?id=58494)

    PowerBI Desktop is a desktop application for authoring PowerBI reports. The desktop app is free to use. PowerBI Desktop is required for the intitial template setup. After setup, you can either run the dashboard locally in the desktop app or publish to the PowerBI SaaS offering ("Known as the PowerBI Service").

    Please note that PowerBI Desktop requires a Windows operating system. Once the report is published to the PowerBI Service, it can be accessed on any operating system.
  </Accordion>
</AccordionGroup>

### PowerBI Service

<AccordionGroup>
  <Accordion title="PowerBI Free Account" icon="github">
    A PowerBI free account provides limited access to the PowerBI Service. With a free account, users can publish reports from PowerBI Desktop to "My Workspace", which is only accessible by that user. 

    <Warning>
      A PowerBI free account will not allow you to share reports with any other user.
    </Warning>
  </Accordion>
  <Accordion title="PowerBI Pro" icon="palette">
    A PowerBI Pro license is a per-user license that provides additonal functionality to the PowerBI Service. The most important feature of a Pro license is **the ability to share reports with other users.**

    <Warning>
      Please note that a license is required for any user viewing or authoring a report.
    </Warning>
  </Accordion>
</AccordionGroup>
